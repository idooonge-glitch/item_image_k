# item_image_k

쿠팡 상품 스크린샷 -> Threads 반자동 게시용 이미지 저장소 겸 트리거입니다.
(계정 K 전용 — `item_image`와 동일한 코드를 공유하지만 독립된 계정/시크릿으로 동작합니다.)

## 쓰는 법

1. 이 저장소의 [`uploads/`](./uploads) 폴더에 쿠팡 상품 스크린샷을 업로드합니다.
2. 몇 초~1분 내로 이 저장소에 새 [이슈(Issue)](../../issues)가 자동으로 생성됩니다.
3. 스크린샷과 맞는 상품 번호를 **이슈에 댓글로** 남깁니다 (예: `3`).
   - 번호가 아니라 다른 텍스트를 남기면 그 텍스트로 다시 검색합니다.
   - 검색으로 안 나오면 쿠팡 상품 URL을 댓글로 붙여넣어도 됩니다.
4. 번호/URL을 남기면 자동으로 딥링크가 생성되고 Threads에 게시된 뒤 이슈가 닫힙니다.

실제 파이프라인 코드는 별도 private 저장소(`coupang-threads-images-manual`)에 있고,
이 저장소는 GitHub Actions가 그 코드를 가져와 실행하는 트리거 겸 이미지 호스팅 역할만 합니다.
